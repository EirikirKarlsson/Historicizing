HTML 1

HTML Element & Attribute Developments:
  • <dir> adopted.

  • <isindex> adopted.

  • <listing> adopted.

  • <menu> adopted.


HTML 1.0 launched in 1991 along with the Web available to the public. This version of HTML developed as a hybrid of the earlier Standard Generalized Markup Language (SGML) developed in the mid‐80's. Tim Berners-Lee, while working at CERN, proposed this standardized hypertext system to simplify communication between members of global CERN teams. The original specifications created by Berner‐Lee only included 20 elements whose primary function was to allow document access to programmers.

HTML 2

HTML 2.0 was released in 1994 as the first official standard for HTML. It was during this period that personal and commercial web use explodes. While HTML 2.0 was the standard additional web features developed including Client-Side Image Maps which were document elements allowing the user to select an area of an image to access different network resources. Tables also were supported to improve the presentation and control over that type of information. Just prior to the release of HTML 2.0 and during its standard a number of web-browers developed including: Cello, Arena, Lyns, tkWWW, and Mosaic (which later became Netscape).

HTML 4.1

HTML Element & Attribute Developments:
  • <applet> deprecated, replaced by <object>.

  • <basefont> deprecated, replaced by font properties in CSS stylesheets.

  • <center> deprecated, replaced by "text‐align"property in CSS stylesheets.

  • <dir> deprecated, replaced by <ul> element.

  • <font> deprecated, replaced by text type properties (i.e., "p"), in CSS stylesheets.

  • <frame>, <frameset>, and <noframes> adopted (never deprecated), replaced by <iframe>.

  • <isindex> deprecated, replaced by <form>.

  • <menu> deprecated, not replaced.
