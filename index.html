<basefont>

As an example, <basefont> originated with HTML 3.2 for the purpose of designating the font style, size, and color for an entire website. The code employed three attributes: color, face, and size. Example of the code in use:

<head>
<basefont face = "Helvetica" size="12 px" color="black" />
<head>

<basefont> was deprecated in HTML 4.01 in favor CSS&aposs greater ability and specificity when formatting. <basefont> became obsolete with HTML 5. As a result, <basefont> is no longer supported by Internet Explorer, Firefox, Safari, Opera, or Chrome.
